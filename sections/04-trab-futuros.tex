\section{Trabalhos Futuros}
\label{sec:resultados}

O presente projeto aborda um tema relevante para o contemporâneo em um contexto atrativo para o âmbito da tecnologia. Além disso, apresenta um nível de maturidade médio e bastante robustez no quesito de tecnologias desenvolvidas. Desta forma, os trabalhos futuros para o mesmo consistem em:

\begin{enumerate}
    \item Ajustar os detalhes da instalação;
    \item Tratar erros de medição;
    \item Integrar o \textit{dashboard} junto ao \textit{ManagerSAFE}
    \item Avaliar o dispositivo;
    \item Avaliar o \textit{dashboard};
    \item Avaliar o \textit{ManagerSAFE};
\end{enumerate}

Além disso, o projeto SAFE faz parte de uma família de dispositivos, logo, após a conclusão do desenvolvimento acerca deste projeto pretende-se retomar o desenvolvimento do OxímetroIoT, implementando atualizações parecidas tendo em vista a arquitetura semelhante entre os projetos, mas adequando-se as particularidades de cada sistema.
