A pandemia de COVID-19 colocou em evidência a importância do controle de biossegurança em edifícios, para conter o contágio por agentes biológicos, bem como exigiu a criação de novas soluções para lidar com estes fatores. Por isto, o projeto SAFE-UFRJ pretende desenvolver um dispositivo IoT de baixo custo para realizar o monitoramento destes indicadores e facilitar o controle e manutenção dos espaços, através de um sistema para o gerenciamento de instalações, que permita a visualização do estado de cada sala que possua um dispositivo, tal como possibilite ao usuário requisitar a manutenção do ambiente.
